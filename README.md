# 🌐 CCNA-Labs

A collection of Cisco CCNA practice labs built using **Cisco Packet Tracer** and **GNS3**. This repository documents hands-on networking configurations, troubleshooting exercises, and topology designs aligned with the **Cisco CCNA 200-301** exam objectives.

---

## 📚 Topics Covered

| #  | Topic                        | Description                                              |
|----|------------------------------|----------------------------------------------------------|
| 01 | VLANs                        | VLAN creation, port assignment, and trunk configuration  |
| 02 | Inter-VLAN Routing           | Router-on-a-stick and Layer 3 switch configurations      |
| 03 | OSI Model                    | Protocol analysis and layer-by-layer troubleshooting     |
| 04 | Subnetting & IP Addressing   | VLSM, CIDR, and address planning exercises               |
| 05 | Routing Protocols            | Static routing, RIP, OSPF, and EIGRP configurations      |
| 06 | STP & EtherChannel           | Spanning Tree Protocol and link aggregation              |
| 07 | NAT & PAT                    | Address translation on Cisco routers                     |
| 08 | ACLs                         | Standard and extended access control lists               |
| 09 | DHCP & DNS                   | Server configuration and client troubleshooting          |
| 10 | WAN & VPN Basics             | PPP, GRE tunnels, and site-to-site VPN concepts          |

## 🛠️ Tools Used

- **Cisco Packet Tracer** – Network simulation
- **GNS3** – Advanced emulation (optional)
- **Wireshark** – Packet capture and analysis

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/pruthvirajs2004/CCNA-Labs.git
   ```
2. Open `.pkt` files with **Cisco Packet Tracer** (version 8.x recommended).
3. Follow the `README.md` inside each lab folder for step-by-step instructions.
