# 🌐 CCNA-Labs

A collection of Cisco CCNA networking labs built using Cisco Packet Tracer and GNS3. This repository showcases hands-on networking configurations, troubleshooting exercises, topology designs, and implementation scenarios aligned with the Cisco CCNA 200-301 certification objectives.

## 📖 Overview

This repository serves as a practical portfolio of my networking knowledge and lab experience. Each lab is designed to simulate real-world networking environments and strengthen skills in routing, switching, network services, security, and troubleshooting.

The repository includes topology diagrams, device configurations, verification commands, and troubleshooting methodologies commonly used in enterprise networks.

## 📚 Topics Covered

| #  | Topic                      | Description                                                |

| 07 | NAT & PAT                  | Address translation and internet connectivity scenarios    |
| 08 | ACLs                       | Standard and Extended Access Control Lists                 |
| 09 | DHCP & DNS                 | Automated IP allocation and name resolution services       |
| 10 | WAN & VPN Basics           | PPP, GRE Tunnels, and VPN fundamentals                     |

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/pruthvirajs2004/CCNA-Labs.git
```

2. Open `.pkt` files using Cisco Packet Tracer (version 8.x or later).

3. Navigate to the desired lab folder.

4. Follow the instructions provided in the respective `README.md` file.

5. Configure devices, verify connectivity, and practice troubleshooting scenarios.

## 📋 Common Verification Commands

```cisco
show ip interface brief
show running-config
show vlan brief
show interfaces trunk
show ip route
show ip ospf neighbor
show ip eigrp neighbors
show standby brief
show access-lists
```
